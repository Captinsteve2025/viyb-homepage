## üöÄ Quick Start Paths

### Path 1: Shortcode (2 Minutes)
1. Upload `viyb-roi-calculator-shortcode.php` to WordPress
2. Activate plugin
3. Add `[viyb_roi_calculator]` to page
4. Done!

**Best for:** WordPress sites with admin access

---

### Path 2: iframe (5 Minutes)
1. Host `charter-roi-calculator-embed.html` on your server
2. Add responsive iframe code to WordPress
3. Publish
4. Done!

**Best for:** Quick embeds, external sites, WordPress.com

---

### Path 3: Direct HTML (10 Minutes)
1. Copy content from `charter-roi-calculator-embed.html`
2. Paste into WordPress HTML block
3. Adjust styling if needed
4. Publish
5. Done!

**Best for:** Complete control, no iframe needed
